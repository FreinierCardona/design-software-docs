# Glosario

> Estado: 🟢 Completo | Última actualización: 2026-06-29
>Autor: Equipo de Arquitectura | | Equipo: Sistema de Gestión de Horarios SENA |

| Término | Definición | Contexto |
|---------|------------|----------|
| Ficha | Grupo de aprendices matriculados en un mismo programa de formación y que comparten un proceso formativo durante un periodo determinado. Cada ficha posee un número único de identificación dentro del SENA. | SENA |
| Jornada | Franja horaria oficial en la que se desarrolla la formación. Puede corresponder, por ejemplo, a jornada diurna, nocturna o mixta, según la programación institucional. | SENA |
| Horario | Planificación organizada de sesiones de formación que asigna fechas, horas, ambientes e instructores para el desarrollo de una ficha o grupo de fichas. | Dominio |
| Instructor | Servidor público o contratista responsable de orientar el proceso de formación profesional integral de los aprendices y ejecutar las actividades académicas asignadas. | SENA |
| Aprendiz | Persona matriculada en un programa de formación del SENA que participa en las actividades académicas, prácticas y de evaluación definidas en su proceso formativo. | SENA |
| Programa de Formación | Oferta académica diseñada por el SENA que define las competencias, resultados de aprendizaje y duración del proceso formativo. | Académico |
| Competencia | Conjunto de conocimientos, habilidades y actitudes que el aprendiz debe desarrollar durante el proceso de formación. | Académico |
| RAP (Resultado de Aprendizaje) | Resultado esperado que evidencia el logro de una competencia específica dentro de un programa de formación. | Académico |
| Ambiente de Formación | Espacio físico o virtual destinado al desarrollo de las actividades de formación, equipado con los recursos necesarios para cada competencia. | Infraestructura |
| Recurso | Bien físico o tecnológico utilizado dentro de un ambiente de formación para apoyar el proceso académico, como computadores, proyectores, herramientas o mobiliario. | Infraestructura |
| Reserva | Asignación temporal de un ambiente o recurso para una actividad académica o administrativa durante un periodo específico. | Infraestructura |
| Disponibilidad | Estado que indica si un ambiente o recurso puede ser utilizado en una fecha y horario determinados. | Infraestructura |
| Sesión de Clase | Unidad de programación que representa una actividad formativa específica dentro de un horario, asociando instructor, ficha, ambiente y franja horaria. | Horarios |
| Franja Horaria | Intervalo de tiempo predefinido utilizado para estructurar la programación académica institucional. | Horarios |
| Conflicto de Horario | Situación en la que dos o más asignaciones coinciden sobre el mismo instructor, ambiente, recurso o ficha en un mismo intervalo de tiempo. | Horarios |
| Microservicio | Servicio independiente que implementa un conjunto específico de funcionalidades del sistema y administra de forma autónoma su propia base de datos. | Arquitectura |
| Clean Architecture | Patrón arquitectónico adoptado por el proyecto para separar responsabilidades, facilitar el mantenimiento y reducir el acoplamiento entre las diferentes capas del sistema. | Arquitectura |
| Auditoría | Registro histórico de las operaciones realizadas sobre la información del sistema para garantizar trazabilidad, seguimiento y control. | Plataforma |
| Notificación | Mensaje generado por el sistema para informar eventos relevantes, alertas o cambios relacionados con la programación académica. | Plataforma |
