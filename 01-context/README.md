## Contenido

Esta sección reúne la documentación necesaria para comprender el contexto funcional, institucional y conceptual del Sistema de Gestión de Horarios SENA (SGH SENA).

Aquí se documenta el problema que busca resolver el proyecto, su alcance general, el entorno institucional donde será implementado y el lenguaje común utilizado durante el análisis, diseño, desarrollo y mantenimiento del sistema.

Toda la documentación contenida en esta carpeta constituye el punto de partida para comprender el dominio del negocio antes de revisar la arquitectura, los microservicios o los modelos de datos.

## Objetivo de esta documentación

- Comprender el contexto del proyecto antes de iniciar el desarrollo.
- Unificar el entendimiento entre analistas, desarrolladores y demás interesados.
- Definir un lenguaje común para todos los módulos del sistema.
- Servir como referencia durante el análisis, diseño, implementación y evolución del SGH SENA.
